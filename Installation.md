Install from Binary Packages
----------------------------

### Ubuntu 14.04 LTS (Trusty Tahr)

    $ sudo apt-add-repository -y 'deb http://apt.conreality.org/ubuntu trusty main'
    $ sudo apt-get update
    $ sudo apt-get install -y conreality
